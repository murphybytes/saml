Single Sign On Test Page

This page is used to test single sign on identity providers. The Relay URL field contains the URL of the service provider resource to invoke after authentication with the IDP. It defaults to this page. Click Request Authorization to trigger the authorization process. The browser's javascript console may contain useful debugging information. The SAML Tracer add-on for Firefox is also useful in diagnosing problems with a particular identity provider.
